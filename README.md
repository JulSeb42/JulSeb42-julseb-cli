-   [React + JS + GraphQL](https://github.com/JulSeb42/JulSeb42-julseb-cli/tree/master/react-js-graphql)
-   [React + TS (client only)](https://github.com/JulSeb42/JulSeb42-julseb-cli/tree/master/react-ts)
-   [React + JS (client only)](https://github.com/JulSeb42/JulSeb42-julseb-cli/tree/master/react-js)

## Use with npx

Run:

```
npx @julseb-lib/julseb-cli <your-project-name>
```

## Install locally

Run:

```
npm install -g @julseb-lib/julseb-cli
```
