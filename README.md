# JulSeb's boilerplates

Repository with all the boilerplates generated by `julseb-cli`.

## Boilerplates

-   [React + Rest API](https://github.com/JulSeb42/JulSeb42-julseb-cli/tree/master/react-rest)
-   [React (client only)](https://github.com/JulSeb42/JulSeb42-julseb-cli/tree/master/react-client)
-   React + GraphQL (coming soon)

## Use with npx

Run:

```
npx julseb-cli <your-project-name>
```

## Install locally

Run:

```
npm install -g @julseb-lib/julseb-cli
```
